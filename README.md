# 🎧 G-Music 

**G-Music** es una pagina interactiva dedicada a la escucha crítica, la alta fidelidad (Hi-Fi) y la divulgación de formatos de audio de alta resolución. 

Este proyecto fue desarrollado para demostrar la implementación de maquetación web moderna, diseño responsivo y manipulación del DOM utilizando tecnologías web estándar sin depender de frameworks externos.

🔗 **[Ver el proyecto en vivo aquí](https://gre3nsn0w.github.io/G-Music/)**

---

## Características Principales:

* **Diseño Oscuro y Elegante:** Interfaz de usuario (UI) construida con CSS puro, utilizando variables de color, Flexbox y CSS Grid para una estructura adaptable y moderna.
* **Totalmente Responsivo:** El diseño se adapta de manera fluida a dispositivos móviles, tablets y monitores de escritorio mediante el uso de `@media queries`.
* **Galería Infinita Animada:** Implementación de un carrusel de imágenes con desplazamiento continuo (`infinite scrolling`) utilizando animaciones `@keyframes` de CSS.
* **Reproductor Interactivo (ECMAScript):** Sección dinámica que compara formatos de audio (FLAC, DSD, MP3). Utiliza Vanilla JavaScript para actualizar el DOM en tiempo real, modificando estadísticas, descripciones e inyectando un reproductor de audio nativo de HTML5 según la selección del usuario.
* **Formulario de Registro Personalizado:** Maquetación completa de un formulario de captura de datos con validaciones básicas de HTML5 y retroalimentación interactiva.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica.
* **CSS3:** Estilos, variables nativas, animaciones y diseño responsivo.
* **JavaScript:** Lógica interactiva y manipulación dinámica.
